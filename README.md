# Cloud Native AI Lab

Hands-on lab materials for the **KubeCon EU 2026** session on Kubernetes and AI workloads.

## Overview

This repository contains code, configurations, and exercises for running AI/ML workloads on Kubernetes using cloud-native tooling.

## Prerequisites

- Kubernetes cluster (v1.28+)
- kubectl configured
- Python 3.11+
- Helm 3.x

## Getting Started

```bash
git clone git@github.com:peopleforrester/cloud-native-ai-lab.git
cd cloud-native-ai-lab
```
